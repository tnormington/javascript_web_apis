History

Gain access to various methods and properties surrounding the browser history.

Example

Click this button to go back to a random point in your Window.history object.

This may have unexpected results depending on your browser history.

Code

    
<p>Click this button to go back to a random point in your Window.history object.</p>
<button class="btn btn-danger" id="random-history-button">Go Back In Time</button>
    

    
window.onload = function() {
    var l = history.length;
    var randomHistoryIndex = Math.floor(Math.random() * l) + 1;
    var button = document.getElementById('random-history-button');
    
    button.addEventListener('click', function() {
        console.log(-randomHistoryIndex);
        history.go(-randomHistoryIndex);
    });
};
